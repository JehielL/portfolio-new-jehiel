Get next line | C

2024

"Leer una línea de un fd es demasiado aburrido...¿Por qué no leer múltiples líneas?"
Get next line es un proyecto que nos invita a crear un programa que debe leer una línea, almacenarla en un buffer y devolverla cada vez que se llama la función. Este proyecto nos permite entender mejor la manipulación de archivos, las variables static y la gestión de memoria en C, poniendo a prueba nuestros conocimientos.

Born2BeRoot | Linux

2024

Entre el titulo y la imagen, creo que os he dado ya una pista, en Born2BeRoot el objetivo principal es configurar un sistema operativo desde cero, comprendiendo y aplicando conceptos clave como la gestión de usuarios y permisos, la configuración de un firewall, la instalación y configuración de servicios, y la seguridad básica del sistema.

Ft_printf | Funciones | C

2024

Printf es un proyecto en el que recreas la función printf de la biblioteca estándar de C, una de las más utilizadas para la salida formateada en programas. Este proyecto te ayuda a entender profundamente el manejo de funciones variádicas, el formateo de cadenas, y la manipulación de tipos de datos en C.

So long | Videojuegos | C

2024

Es hora de divertirnos, en este proyecto desarrollas un pequeño juego en 2D, aprendiendo a manejar gráficos, eventos de teclado y ventanas en C usando la biblioteca MinilibX. Pipex | Programación | C

2024

Pipex es un proyecto en el que aprendes a manejar la redirección de entradas y salidas entre procesos en Linux, replicando la funcionalidad de una tubería (pipe). En este proyecto, profundizas en conceptos clave como execve, fork, y especialmente heredoc, que permite redirigir entradas hasta una palabra clave, este proyecto sube el grado de dificultad y es la entrada para la gran Minishell.

Push Swap | Algoritmia | C

2024

Es un proyecto en el que desarrollas un algoritmo eficiente para ordenar una pila de números utilizando un conjunto limitado de operaciones.
